# プライバシーポリシー

本プライバシーポリシーは、スマートフォンアプリ「One Life 80 Years」（以下、「本アプリ」）における、ユーザー情報の取り扱いについて説明するものです。

## 1. 収集する情報
本アプリでは、以下の情報を取得する場合があります。

- 広告配信のための広告識別子（Google Advertising ID 等）
- 端末情報（OS バージョン、端末の種類など）
- アプリの利用状況（表示回数、操作履歴などの匿名の統計情報）

これらの情報は、個人を特定する目的では使用されません。

## 2. 情報の利用目的
取得した情報は、以下の目的で利用されます。

- 広告の表示および最適化
- アプリの品質向上および不具合修正
- 利用状況の統計分析

## 3. 広告について
本アプリでは、第三者配信の広告サービス（Google AdMob）を利用しています。

AdMob は、ユーザーの興味に応じた広告を表示するために、広告識別子等の情報を使用する場合があります。

広告に関する詳細は、以下の Google のプライバシーポリシーをご確認ください。  
https://policies.google.com/privacy

## 4. 第三者への情報提供
本アプリは、法令に基づく場合を除き、取得した情報を第三者に提供することはありません。  
ただし、広告配信および分析のため、Google などの外部サービスに情報が送信される場合があります。

## 5. 児童のプライバシー
本アプリは、13歳未満の児童から意図的に個人情報を収集することはありません。

## 6. プライバシーポリシーの変更
本ポリシーは、必要に応じて予告なく変更されることがあります。  
変更後の内容は、本ページにて公表します。
